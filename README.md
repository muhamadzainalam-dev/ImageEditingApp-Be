# ImageEditingApp – Backend

Backend processing layer for image tooling (compression, resize, conversion-style workflows) and file handling—built for fast, optimized outputs.

> 🔗 **Frontend Repo:** `muhamadzainalam-dev/ImageEditingApp-Fe`  
> `https://github.com/muhamadzainalam-dev/ImageEditingApp-Fe`

---

## 💼 Skills & Tech Used

- **Server:** Node.js, Express
- **Image Processing:** Sharp
- **Uploads:** Multer
- **Compression/Archiving:** Archiver
- **HTTP Clients:** Axios, node-fetch
- **Parsing/Scraping:** Cheerio
- **Middleware:** CORS
- **Utilities:** `ytdl-core`, `play-dl`, `path`

---

## 🚀 Run Locally

```bash
git clone https://github.com/muhamadzainalam-dev/ImageEditingApp-Be.git
cd ImageEditingApp-Be
npm install
npm start
```
